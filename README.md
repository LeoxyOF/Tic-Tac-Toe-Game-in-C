# Tic-Tac-Toe-Game-in-C

--English--

This is a two-player Tic Tac Toe game developed in C, played directly in the terminal. The program randomly selects which player starts, validates user input, detects wins and draws, and keeps track of the score across multiple rounds.

Features:

Random selection of the first player (X or O)

Interactive 3×3 board display

Input validation for row and column choices

Automatic win detection (rows, columns, and diagonals)

Draw detection

Score tracking across multiple games

Option to play again or exit

--Português--

Este é um jogo da velha para dois jogadores, desenvolvido em C e jogado diretamente no terminal. O programa seleciona aleatoriamente o jogador inicial, valida a entrada do usuário, detecta vitórias e empates e registra a pontuação ao longo de várias rodadas.

Recursos:

Seleção aleatória do primeiro jogador (X ou O)

Exibição interativa de um tabuleiro 3x3

Validação de entrada para escolha de linhas e colunas

Detecção automática de vitórias (linhas, colunas e diagonais)

Detecção de empate

Registro de pontuação ao longo de várias partidas

Opção de jogar novamente ou sair
